## 🚀 Featured Project

### 🖐️ Hand Gesture Recognition System

> Real-time gesture detection using a live webcam feed — no hardware, just pure Python magic.

- 📸 Built with **Python + OpenCV + MediaPipe** to detect and classify hand landmarks in real time
- ✋ Tracks **finger positions and hand movements** with high accuracy
- ⚡ Gesture-based controls that respond instantly — demonstrating practical Computer Vision
- 🧠 Lays the groundwork for touchless UI, accessibility tools, and human-computer interaction
